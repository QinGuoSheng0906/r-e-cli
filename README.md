### 目录介绍：
```
       pubilc: html 模板
          src: 工程目录
          api: axios 封装的目录
       assets: 全局资源目录 图片/字体图标/样式
   components: 公共组件
       config: 公共配置
        mocks: 模拟数据   
        pages: 页面  
        redux: 状态管理
      routers: 路由配置
        utils: 工具  
```
